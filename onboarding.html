Welcome to Event Alert Extension

Thank you for installing Event Alert Extension! This guide will help you get started with monitoring websites for upcoming events and receiving timely notifications.

1Add Your First Event to Monitor

Click on the extension icon in your browser toolbar and go to the "Add Event" tab:

  1. Enter the full URL of the website containing the event information (e.g., https://example.com/events)
  2. Choose to auto-detect dates on the page or set a date manually
  3. If auto-detecting, click "Detect Dates" and select the event date on the page
  4. Add an optional event name for easier identification
  5. Select when you want to receive notifications
  6. Click "Add Event" to start monitoring

FAQ

Q: How often will the extension check for upcoming events?

A: The extension performs a daily check to see if any events are approaching their notification dates.

Q: Will this slow down my browser?

A: No, the extension is designed to be resource-efficient. Checks happen on a schedule in the background, not continuously.

Q: Is there a limit to how many events I can monitor?

A: Since all data is stored locally, you can monitor as many events as you want, but keeping the list manageable will make it easier to track your important events.

Q: What happens if I'm not using my browser when an event notification is triggered?

A: Chrome will handle the notification based on your system settings. On most systems, you'll see the notification in your operating system's notification center.

Q: Can I sync my events across multiple devices?

A: The extension uses Chrome's sync storage, so if you're signed into Chrome on multiple devices, your events should sync automatically.

4Managing Your Events

The "My Events" tab shows all the events you're currently monitoring. For each event, you can:

  • See how many days remain until the event
  • Toggle monitoring on/off using the switch
  • View details about the event URL and date
  • Edit the event settings
  • Delete events you no longer want to monitor
Tip: The color-coded indicators show how close your events are:
  • Today! - Event is happening today
  • Tomorrow! - Event is happening tomorrow
  • 3 days left - Event is approaching soon
  • 10 days left - Event is in the future
  • Event has passed - Event date has gone by

How Event Reminders Work

When you add an event to monitor, the extension will:

  1. Store the event information locally on your device
  2. Check daily to see if any of your events are approaching their notification dates
  3. Send browser notifications based on your preferences (1 week, 3 days, and/or 1 day before)
  4. Show color-coded indicators in the extension to help you keep track of upcoming events
Note: This extension stores all data locally on your device. Your event information is not uploaded to any servers or shared with anyone.

2Understanding Date Detection

When you click "Detect Dates," a new tab will open with the target website. Here's how to use the date detection tool:

  1. The extension will automatically scan the page for dates
  2. All detected dates will be highlighted in yellow
  3. Click on the date that corresponds to your event
  4. The date will be automatically saved and the tab will close
Tip: If the extension doesn't detect the correct date, you can always set it manually using the date picker option.

3Configure Notification Settings

Go to the "Settings" tab to configure how you want to be notified about upcoming events:

  1. Enable or disable browser notifications
  2. Choose your preferred notification schedule
Tip: For important events, we recommend setting notifications at multiple intervals (1 week, 3 days, and 1 day before).
