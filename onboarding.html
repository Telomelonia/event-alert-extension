Welcome to Event Alert Extension

Thank you for installing Event Alert Extension! This guide will help you get started with monitoring websites for changes and receiving notifications.

1Add Your First URL to Monitor

Click on the extension icon in your browser toolbar and:

  1. Enter the full URL of the website you want to monitor (e.g., https://example.com/events)
  2. If you want to monitor only a specific section of the page, use the "Pick from page" button to select that element
  3. Choose how frequently you want to check for changes (hourly, daily, or weekly)
  4. Click "Add URL" to start monitoring

2Configure Notification Settings

Go to the "Settings" tab to configure how you want to be notified about changes:

  1. Choose whether to receive browser notifications
  2. Select notification frequency (immediate, daily digest, or weekly digest)
  3. Click "Save Settings" to apply your preferences
Tip: For important websites, choose "immediate" notifications. For less critical websites, consider using digests to reduce the number of notifications you receive.

3Managing Your Monitored URLs

The "My URLs" tab shows all the websites you're currently monitoring. For each URL, you can:

  • Toggle monitoring on/off using the switch
  • View details about the URL, selector, and last check time
  • Edit the monitoring settings
  • Delete URLs you no longer want to monitor

Understanding Element Selection

When you click "Pick from page," a new tab will open with the target website. Here's how to use the selector tool:

  1. Hover over different elements on the page. They will be highlighted as you move your mouse.
  2. The selector tool will show you the CSS selector for the current element.
  3. When you find the section you want to monitor, click on it to select.
  4. The selector will be automatically filled in the extension form.
Tip: Select the most specific container that holds the content you want to monitor. For example, if you're watching for new products, select the product listing container rather than the entire page.

FAQ

Q: How often will the extension check for changes?

A: The extension will check hourly, daily, or weekly, depending on your selection for each URL.

Q: Will this slow down my browser?

A: No, the extension is designed to be resource-efficient. Checks happen on a schedule in the background, not continuously.

Q: Is there a limit to how many URLs I can monitor?

A: There's no hard limit, but monitoring too many URLs might use more resources. Start with a few important URLs and add more as needed.

Q: What happens if I'm not using my browser when a change is detected?

A: You'll see a notification the next time you open your browser if changes were detected while you were away.
